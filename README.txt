GitHub Profile Viewer

Overview

This is a basic web page that allows users to view public repositories of a GitHub user. The page displays the user's profile information along with a list of their public repositories. Users can customize the number of repositories displayed per page using a dropdown menu.



Features

Profile Information: The user's GitHub profile picture, login, and name (if available) are displayed at the top of the page.



Public Repositories: The page lists the user's public repositories. Each repository entry includes the repository name and description (if available).



Pagination: Users can choose the number of repositories to display per page using a dropdown menu. Options range from 10 to 100 repositories per page.



Usage

Open the HTML file (index.html) in a web browser.



The user's GitHub profile information and a list of public repositories will be displayed.



Use the dropdown menu labeled "Show repositories per page" to select the desired number of repositories to display per page.



The page will update to show the selected number of repositories.



Technologies Used

HTML

JavaScript (Fetch API)


How to Run

Simply open the index.html file in a web browser to view the GitHub profile and repositories.
